# AI Shopping Mall Project

## Description
Full-stack shopping mall web application with an AI-based recommendation system.

## Project Info
Period: 2025.09 – 2025.12  
Type: Team Project

## Tech Stack
Frontend
- React

Backend
- Node.js
- Express

AI
- Python

Database
- MySQL 
## Features

### Shopping System
- Browse clothing by category
- Filter by gender and style

### Context-based Recommendation
- Weather / date / location based clothing suggestions
- AI recommendation based on situation and occupation

### AI Interaction
- Chat with AI for outfit recommendations

### Social Communication
- Real-time communication between current users

### User System
- User signup
- Login / Logout
- Session management

### Shopping Functions
- Add items to cart
- Manage shopping cart
